# 🔑 如何找到或创建数据库用户密码

## 📋 两种密码的区别

### 1. MongoDB Atlas 账户密码
- **用途**：登录 MongoDB Atlas 网站（https://cloud.mongodb.com）
- **位置**：你注册时设置的密码
- **用途**：登录管理界面

### 2. 数据库用户密码
- **用途**：应用程序连接数据库时使用
- **位置**：在 Database Access 中创建的用户密码
- **用途**：连接字符串中的密码

## 🔍 如何找到数据库用户密码

### 方法一：查看现有数据库用户

1. **进入 Database Access**
   - 访问：`https://cloud.mongodb.com/v2#/access/users`
   - 或点击顶部导航 → Security → Database Access

2. **查看用户列表**
   - 找到用户名为 `html-tools-admin` 的用户
   - 如果能看到密码（通常显示为 `****`），说明已创建
   - **注意**：MongoDB Atlas 不会显示已保存的密码，只能看到用户名

3. **如果忘记密码**
   - 需要重置密码或创建新用户

### 方法二：创建新的数据库用户（推荐）

如果找不到密码或想创建新用户：

1. **进入 Database Access**
   - 访问：`https://cloud.mongodb.com/v2#/access/users`
   - 或点击顶部导航 → Security → Database Access

2. **创建新用户**
   - 点击 **"Add New Database User"** 或 **"Add Database User"**
   - Authentication Method: 选择 **"Password"**
   - Username: 输入 `aio2admin`（可自定义）
   - Password: 
     - 点击 **"Autogenerate Secure Password"** 生成密码
     - **重要**：立即复制并保存这个密码！
     - 或自己设置一个强密码
   - Database User Privileges: 选择 **"Atlas admin"** 或 **"Read and write to any database"**
   - 点击 **"Add User"**

3. **使用新用户**
   - 用户名：`aio2admin`（或你设置的用户名）
   - 密码：刚才生成或设置的密码

### 方法三：重置现有用户密码

如果用户 `html-tools-admin` 已存在但忘记密码：

1. **进入 Database Access**
   - 访问：`https://cloud.mongodb.com/v2#/access/users`

2. **找到用户**
   - 找到 `html-tools-admin` 用户
   - 点击用户右侧的 **"Edit"** 或 **"..."** 菜单

3. **重置密码**
   - 选择 **"Edit Password"** 或 **"Reset Password"**
   - 生成新密码或设置新密码
   - **重要**：复制并保存新密码！
   - 保存更改

## 🎯 推荐操作

### 方案 A：使用现有用户（如果知道密码）

如果你记得 `html-tools-admin` 的密码：
- 直接使用该密码生成连接字符串

### 方案 B：创建新用户（推荐，更清晰）

创建一个专门用于 AIO2 项目的用户：

1. 用户名：`aio2admin`
2. 密码：生成强密码并保存
3. 权限：Atlas admin

这样更清晰，便于管理。

## 📝 生成连接字符串

获取到数据库用户密码后，连接字符串格式：

```
mongodb+srv://用户名:密码@cluster0.gpq75zd.mongodb.net/jeyi-toolhub?retryWrites=true&w=majority
```

**使用现有用户 `html-tools-admin`**：
```
mongodb+srv://html-tools-admin:你的密码@cluster0.gpq75zd.mongodb.net/jeyi-toolhub?retryWrites=true&w=majority
```

**使用新用户 `aio2admin`**：
```
mongodb+srv://aio2admin:你的密码@cluster0.gpq75zd.mongodb.net/jeyi-toolhub?retryWrites=true&w=majority
```

## ⚠️ 重要提示

1. **密码不会显示**
   - MongoDB Atlas 不会显示已保存的密码
   - 如果忘记，只能重置或创建新用户

2. **密码需要 URL 编码**
   - 如果密码包含特殊字符（`@`、`#`、`$` 等），需要进行 URL 编码
   - 可以使用脚本自动处理

3. **安全建议**
   - 使用强密码
   - 不要将密码分享给他人
   - 定期更换密码
